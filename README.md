# MoscowTime - iOS приложение

Простое iOS приложение для отображения текущего дня недели и времени в Москве.

## Описание

Приложение показывает:
- День недели на русском языке
- Текущее время в формате HH:mm:ss (московское время)
- Полную дату в формате "день месяц год"

Время обновляется каждую секунду автоматически.

## Требования

- iOS 15.0 или выше
- iPhone 11 или новее (или любой iPhone с iOS 15+)

## Сборка через GitHub Actions

Этот проект настроен для автоматической сборки через GitHub Actions на Mac runner.

### Шаги для сборки:

1. **Создайте репозиторий на GitHub**
   - Создайте новый публичный репозиторий
   - Загрузите все файлы проекта в репозиторий

2. **Запустите сборку**
   - GitHub Actions автоматически запустится при push в main/master ветку
   - Или запустите вручную через вкладку "Actions" → "Build iOS App" → "Run workflow"

3. **Скачайте IPA файл**
   - После завершения сборки перейдите в "Actions"
   - Откройте последний workflow run
   - В разделе "Artifacts" скачайте "MoscowTime-IPA"
   - Распакуйте архив и найдите файл `MoscowTime.ipa`

## Установка на iPhone через Sideloadly

### Подготовка:

1. **Установите iTunes** (не из Microsoft Store!)
   - Скачайте с официального сайта Apple: https://www.apple.com/itunes/download/
   - Это необходимо для подключения iPhone к компьютеру

2. **Скачайте Sideloadly**
   - Перейдите на официальный сайт: https://sideloadly.io/
   - Скачайте версию для Windows
   - Установите программу

### Установка приложения:

1. **Подключите iPhone к компьютеру**
   - Используйте оригинальный USB кабель
   - На iPhone появится запрос "Доверять этому компьютеру?" → нажмите "Доверять"
   - Разблокируйте iPhone (если требуется)

2. **Откройте Sideloadly**
   - Запустите программу Sideloadly на Windows

3. **Выберите IPA файл**
   - Нажмите кнопку "IPA" или перетащите файл `MoscowTime.ipa` в окно Sideloadly

4. **Введите Apple ID**
   - В поле "Apple ID" введите ваш email от Apple ID
   - В поле "Password" введите пароль от Apple ID
   - ⚠️ **Важно**: Sideloadly использует ваш Apple ID для создания бесплатного сертификата разработчика. Это безопасно, но если у вас включена двухфакторная аутентификация, вам может потребоваться создать пароль приложения в настройках Apple ID.

5. **Выберите устройство**
   - Убедитесь, что ваш iPhone отображается в списке устройств
   - Если iPhone не виден, проверьте подключение и доверие компьютеру

6. **Начните установку**
   - Нажмите кнопку "Start"
   - Дождитесь завершения процесса установки

7. **Доверьте разработчика на iPhone**
   - После установки на iPhone откройте: **Настройки → Основные → VPN и управление устройством**
   - Найдите запись с вашим Apple ID email
   - Нажмите на неё и выберите "Доверить [ваш email]"
   - Подтвердите доверие

8. **Готово!**
   - Приложение "Москва" появится на главном экране iPhone
   - Можете запускать его и использовать

## Важные замечания

- **Срок действия сертификата**: Приложения, установленные через Sideloadly с бесплатным Apple ID, требуют переподписи каждые 7 дней. Для этого просто подключите iPhone к компьютеру и запустите Sideloadly снова.

- **Безопасность**: Sideloadly использует официальный метод Apple для подписи приложений. Ваш Apple ID используется только для создания сертификата разработчика.

- **Двухфакторная аутентификация**: Если у вас включена 2FA, создайте пароль приложения в настройках Apple ID: https://appleid.apple.com/ → "Пароли приложений"
