Генератор иконки для MoscowTime

Инструкция:

  1. Нажмите кнопку "Сгенерировать иконку" для создания иконки
  2. Нажмите на нужный размер для скачивания PNG файла
  3. Сохраните файлы в папку MoscowTime/Assets.xcassets/AppIcon.appiconset/
  4. Переименуйте файлы согласно размерам (например, icon-40@2x.png, icon-60@3x.png и т.д.)
