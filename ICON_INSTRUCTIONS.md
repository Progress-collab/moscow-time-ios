# Инструкция по созданию иконок для приложения

## Быстрый способ (рекомендуется)

1. Откройте файл `ICON_GENERATOR.html` в браузере
2. Нажмите кнопку "Сгенерировать все размеры"
3. Все PNG файлы будут автоматически скачаны
4. Скопируйте все скачанные файлы в папку `MoscowTime/Assets.xcassets/AppIcon.appiconset/`

## Альтернативный способ (вручную)

Если генератор не работает, вы можете создать иконки вручную:

### Требуемые размеры:

- `icon-20@2x.png` - 40x40 пикселей
- `icon-20@3x.png` - 60x60 пикселей  
- `icon-29@2x.png` - 58x58 пикселей
- `icon-29@3x.png` - 87x87 пикселей
- `icon-40@2x.png` - 80x80 пикселей
- `icon-40@3x.png` - 120x120 пикселей
- `icon-60@2x.png` - 120x120 пикселей
- `icon-60@3x.png` - 180x180 пикселей
- `icon-76@1x.png` - 76x76 пикселей
- `icon-76@2x.png` - 152x152 пикселей
- `icon-83.5@2x.png` - 167x167 пикселей
- `icon-1024.png` - 1024x1024 пикселей

### Дизайн иконки:

Иконка должна иметь:
- Градиентный фон (от фиолетового #667eea к пурпурному #764ba2)
- Символ часов с белыми стрелками
- Букву "М" внизу (белая)

### Инструменты для создания:

- **Онлайн**: Canva, Figma, Photopea
- **Десктоп**: Photoshop, GIMP, Paint.NET
- **Простой вариант**: Используйте ICON_GENERATOR.html

## После создания иконок:

1. Убедитесь, что все файлы находятся в папке `MoscowTime/Assets.xcassets/AppIcon.appiconset/`
2. Запустите сборку через GitHub Actions
3. Установите новую версию приложения на iPhone

## Проверка:

После установки приложения на iPhone, иконка должна отображаться на главном экране вместо стандартной иконки iOS.
